# Community Outreach v1

Premium, mobile-first local trust and customer intake landing page for homeowners and local customers.

## Project scope

This project is for local service connection and intake workflows.
It is **not** related to NASA, astronomy, solar flares, DONKI, or space weather.

## Tech stack

- React + Vite
- Tailwind CSS

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Current baseline sections

- Hero section
- Trust explanation section
- Service category cards
- Customer intake form with client-side required validation
- Submit confirmation message
- Placeholder section for future Google Form embed or Google Sheets/webhook integration
