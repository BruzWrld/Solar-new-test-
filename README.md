# Solar-new-test-

This repository provides a simple script to retrieve recent solar event information from NASA's DONKI API.

## Setup

This script requires Python 3 and the `requests` library. Install dependencies with:

```bash
pip install -r requirements.txt
```

## Usage

Run the script to fetch the latest solar notifications:

```bash
python fetch_solar_info.py
```

The script prints summaries of recent solar events such as coronal mass ejections or solar flares.
